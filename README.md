
# SQL Practice Queries

## 📌 Description
This repository contains SQL queries from basic to intermediate level.  
It is created to practice and understand core Database Management System (DBMS) concepts.

## 🚀 Topics Covered
- Basic Queries (SELECT, INSERT, UPDATE, DELETE)
- Conditions (WHERE, AND, OR, LIKE, BETWEEN)
- Aggregate Functions (COUNT, SUM, AVG, MAX, MIN)
- GROUP BY and HAVING
- Joins (INNER JOIN)
- Subqueries
- Table Creation and Constraints

## 🛠️ Technologies Used
- SQL
- DBMS Concepts

## 📂 Repository Structure
- `01_Create_Insert.sql` → Database & table creation + data insertion  
- `02_Select.sql` → Basic SELECT queries  
- `03_Conditions.sql` → Filtering queries  
- `04_Aggregate.sql` → Aggregate functions  
- `05_GroupBy.sql` → GROUP BY & HAVING  
- `06_Joins.sql` → JOIN operations  
- `07_Update_Delete.sql` → UPDATE & DELETE  
- `08_Subqueries.sql` → Subqueries  

## ▶️ How to Use
1. Open any SQL tool (MySQL / SQLPlus)
2. Run the queries step by step
3. Observe the output

## 🎯 Purpose
The main goal of this repository is to improve SQL skills and understand how databases work in real-world applications.
